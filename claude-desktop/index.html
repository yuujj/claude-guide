시작

시작하기

macOS · Windows 웹 기능 포함
1
Claude Desktop 다운로드

Anthropic 공식 사이트(claude.ai/download)에서 macOS 또는 Windows 설치 파일을 내려받아 실행합니다.

2
계정 로그인

앱을 열면 Anthropic 계정으로 로그인합니다. claude.ai와 같은 계정을 사용합니다.

3
탭 선택 후 시작

Chat, Code, Cowork 중 원하는 탭을 선택합니다. 처음이라면 Chat 탭부터 시작하세요.

세팅

나만의 세팅하기

클로드 데스크탑은 웹의 모든 세팅에 더해, 내 환경과 직접 연결하는 방법을 제공합니다.

세팅방법난이도
Chat 세팅 (프로젝트·스타일·메모리) 웹과 동일 — 웹 가이드 참조 쉬움
커넥터 설정 → 연결 → 서비스 선택 → 승인 쉬움
데스크탑 확장 (.mcpb) .mcpb 파일 더블클릭 → 설치 쉬움
MCP 서버 직접 설정 claude_desktop_config.json 편집 고급

MCP 서버를 직접 설정하려면 claude_desktop_config.json 파일을 편집합니다.

json · claude_desktop_config.json 
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "ghp_여기에_토큰_입력" }
    }
  }
}

파일 위치:
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

탭

세 가지 탭

클로드 데스크탑은 Chat, Code, Cowork 세 탭을 통해 서로 다른 작업을 지원합니다.

웹 기능 포함

웹(claude.ai)의 모든 기능을 데스크탑 앱에서 그대로 사용합니다. 프로젝트, 아티팩트, 메모리, 웹 검색이 모두 동작합니다.

Chat 기능 상세는 → 클로드 웹 가이드

Pro 이상 클로드 코드와 동일

클로드 코드의 GUI 버전입니다. 프로젝트 폴더를 열거나 끌어다 놓으면 AI가 파일을 직접 읽고, 수정하고, git 작업을 처리합니다. 터미널 없이 코드 작업을 맡길 수 있습니다.

Code 탭 기능 상세는 → 클로드 코드 가이드

Pro 이상 고급

AI가 화면을 직접 보고 마우스와 키보드로 앱을 조작합니다. 반복적인 GUI 작업 — 폼 작성, 스크린샷 캡처, 특정 앱 조작 — 을 자동화합니다.

1
Cowork 탭 선택

앱 상단에서 Cowork 탭을 선택합니다.

2
작업 지시

"Slack에서 특정 채널의 메시지를 스프레드시트로 정리해줘"처럼 원하는 작업을 자연어로 입력합니다.

3
AI 자동 실행

AI가 화면을 공유받아 직접 앱을 조작하며 작업을 수행합니다.

기능

핵심 기능

클로드 데스크탑에서만 사용할 수 있는 주요 기능입니다.

AI가 화면을 보고 마우스/키보드로 앱을 조작하는 기능입니다. 특정 소프트웨어가 API를 제공하지 않아도, 사람처럼 GUI를 통해 작업할 수 있습니다. 현재 macOS에서 주로 사용 가능합니다.

Slack, Google Drive, Notion 같은 서비스를 클릭 몇 번으로 연결합니다. 설정 → 연결에서 원하는 서비스를 선택하고 승인하면 됩니다. 코드나 JSON 설정 없이 GUI로 가능합니다.

Model Context Protocol로 외부 도구를 Claude에 연결합니다. 커넥터가 없는 서비스(사내 DB, 커스텀 API 등)도 MCP 서버를 통해 연결할 수 있습니다. claude_desktop_config.json을 편집합니다.

.mcpb 확장자 파일을 더블클릭하면 MCP 서버가 자동으로 설치됩니다. MCP 서버를 직접 설정하지 않고도 사전 패키징된 통합 기능을 사용할 수 있는 방법입니다.
