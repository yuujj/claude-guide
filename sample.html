클로드란?

클로드는 Anthropic이 만든 AI 어시스턴트입니다. 텍스트 작성, 코드 생성, 이미지 분석, 번역 등 다양한 작업을 수행합니다. Constitutional AI라는 고유한 학습 방식으로, 안전하고 유용한 응답을 제공합니다.

모델 패밀리

모델특징적합한 작업
Opus 4.6가장 뛰어난 지능복잡한 분석, 대규모 코딩
Sonnet 4.6속도와 지능의 균형일상 업무, 글쓰기
Haiku 4.5가장 빠른 응답단순 질문, 요약

Constitutional AI란?

일반 AI는 사람이 "좋은 답변/나쁜 답변"을 일일이 판단해서 학습합니다. 반면 클로드는 헌법(원칙 문서)을 기반으로 AI 스스로 답변을 평가하고 개선합니다. 원칙이 명시적이고 투명해서, "왜 이렇게 답하는지" 추적할 수 있습니다.

핵심 능력

능력설명예시
텍스트 생성글쓰기, 요약, 번역보고서 초안, 이메일 작성
코드 작성모든 프로그래밍 언어Python 스크립트, 웹앱
이미지 이해사진/차트/스크린샷 분석그래프 해석, UI 피드백
긴 문맥최대 100만 토큰책 한 권, 대규모 코드
도구 사용외부 서비스 연동웹 검색, 파일, DB 조회

3단계로 시작

지금 바로 클로드를 사용해 보세요

1
claude.ai 접속
브라우저에서 claude.ai에 접속하고 무료 계정을 만드세요.
2
대화 시작
"오늘 회의 내용을 요약해줘"처럼 자연어로 질문하세요.
3
프로젝트 만들기
자주 쓰는 파일과 지침을 프로젝트에 저장하면, 매번 설명할 필요가 없어요.
1
설치
bash 
curl -fsSL https://claude.ai/install.sh | bash
2
프로젝트에서 실행
bash 
cd my-project && claude
3
자연어로 작업 지시
prompt 
"auth 모듈 테스트 작성하고 실행해줘"
