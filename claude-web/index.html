# 나에 대해
저는 스타트업에서 마케팅을 담당하고 있습니다.

# 응답 방식
- 항상 한국어로 답변
- 핵심 먼저, 설명은 간결하게
- 수치 예시나 표 적극 활용
활용

이런 걸 할 수 있어요

질문, 글쓰기, 파일 분석, 인터랙티브 앱 제작까지 — 채팅 하나로 가능합니다.

💬

질문·학습

어려운 개념을 내 수준에 맞춰 설명받고, 후속 질문으로 깊이 들어갑니다.

"양자역학을 중학생 수준으로 설명하고, 이해도 확인할 퀴즈 3개 만들어줘"
✍️

글쓰기·번역

상황과 톤을 주면 초안을 만들어줍니다. 커스텀 스타일로 회사 톤을 고정할 수도 있습니다.

"거래처에 납기 지연 안내 메일 — 정중하지만 단호하게, 3문단으로"
🖼️

이미지·파일 분석

PDF, 엑셀 스크린샷, 영수증 사진을 업로드하면 내용을 읽고 정리합니다.

"이 영수증 10장을 표로 정리하고 월별 지출 카테고리 분석해줘"
⚡

만들기 (아티팩트)

채팅 옆에 실시간으로 프로토타입이 렌더링됩니다. 코드를 몰라도 됩니다.

"연봉 협상 시뮬레이터 — 연봉/세금/실수령액 보여주는 인터랙티브 페이지"
기능

핵심 기능

대화, 파일, 지침을 하나의 작업 공간으로 묶습니다. 지침은 해당 프로젝트의 모든 대화에 자동으로 주입되어 매번 설명을 반복하지 않아도 됩니다. 업무별로 프로젝트를 만들어두면 맥락이 유지됩니다.

코드, 문서, 다이어그램, 인터랙티브 앱을 채팅 옆 패널에 실시간으로 렌더링합니다. "연봉 계산기 만들어줘"라고 하면 바로 옆에서 동작하는 계산기를 볼 수 있습니다.

대화 내용을 자동으로 요약하여 다음 세션에서도 기억합니다. Pro 이상에서 활성화됩니다. "저는 마케터입니다", "저는 한국어만 씁니다" 같은 맥락을 한 번 알려두면 됩니다.

실시간 웹을 검색하여 최신 정보를 참조합니다. 기본 모델의 지식은 학습 시점까지이지만, 웹 검색을 활성화하면 오늘 날씨, 최신 뉴스, 실시간 데이터를 답변에 포함합니다.

설정 → 응답 스타일에서 응답 길이, 형식, 어조를 전역으로 고정합니다. "항상 한국어로, 요점만, 최대 3문장"처럼 한 번 설정하면 모든 대화에 적용됩니다.
